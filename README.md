## Установка зависимостей

1. Установите зависимости:
```bash
py -m pip install -r requirements.txt
```

2. Установите браузеры Playwright:
```bash
py -m playwright install
```

## Запуск тестов из директории tests

```bash
pytest tests/
```

## Запуск тестов c принтами 

```bash
pytest tests/ -s
```
